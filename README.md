# My Dotfiles for FreeBSD

## zsh
1. Copy to ~/.config/
2. Run `getplugins.sh` -> this fetches plugins from GitHub
```bash
./getplugins.sh
```
3. Link ~/.config/zshrc to ~/.zshrc
```bash
ln -s ~/.config/zshrc ~/.zshrc
```
4. (optional) modify aliases or zshrc depending on your needs/tastes
